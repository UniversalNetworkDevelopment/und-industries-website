Security Disclaimer

Effective date: May 19, 2026

1. No Real Security

This site does not provide secure authentication, encryption, or protected data storage of any kind. All login and registration features are demo-only and exist solely to demonstrate user interface flows.

This site is not suitable for storing real credentials, sensitive personal data, financial information, or any content requiring security protection.

2. Plaintext Storage

Any credentials entered on this site — including email addresses and passwords — are stored as plaintext (unencrypted) in your browser's localStorage. There is no hashing, no salting, and no encryption applied to these values at any point.

Do not use a real password from another service on this site.

3. No Backend

There is no server, no database, and no protected session management system behind this site. All data remains on your device in your browser's localStorage. It is not transmitted to any server, synchronized, or backed up. Data can be lost or accessed if your browser storage is cleared or your device is accessed by others.

4. User Responsibility

By using this site's account or login features, you acknowledge that:
