# NEXUS AUDIT: MISSING COMPONENTS

## 1. Missing Ticket Creation UI (Severity: CRITICAL)
- **Missing:** A functional client portal to submit non-test jobs.
- **Explanation:** The current `INTAKE_UI.html` exists in `docs/` but lacks the JavaScript fetch wrapper to securely POST to Supabase while handling authentication.
- **Required Fix:** Wire `INTAKE_UI.html` directly to `@supabase/supabase-js` or a Cloudflare Worker proxy.

## 2. Missing Evidence Viewer (Severity: HIGH)
- **Missing:** Dashboard UI to download and view evidence packs.
- **Explanation:** The Website Manager exists, but there is no UI component in the portal to display the `evidence_url` generated by Qwep.
- **Required Fix:** Add an "Evidence" tab to the Nexus Dashboard.

## 3. Missing Test Mode UI (Severity: MEDIUM)
- **Missing:** Owner-only toggle to inject `ticket_type='test'` jobs.
- **Explanation:** There is currently no UI for the owner to safely trigger the test pipeline without manually writing SQL inserts.
- **Required Fix:** Build a dev-tools panel in the portal.
